Questions fréquentes

Comment débloquer mon compte ?

Si votre compte est bloqué, veuillez contacter immédiatement notre service client au 1-800-123-4567 ou visitez l'une de nos succursales avec une pièce d'identité valide pour vérification.

Comment modifier mes informations personnelles ?

Vous pouvez mettre à jour certaines informations directement depuis la section "Mes informations" de votre espace client. Pour les modifications importantes (nom, date de naissance), veuillez visiter une succursale avec les documents justificatifs nécessaires.

Comment suivre un virement international ?

Les virements internationaux peuvent être suivis dans la section "Historique des transactions". Cliquez sur la transaction pour voir les détails. Les virements SWIFT disposent d'un numéro de référence unique que vous pouvez fournir à notre service client pour un suivi plus précis.

Quels sont les frais pour les virements internationaux ?

Les frais varient selon le pays de destination et le montant transféré. En général, comptez entre 10$ CA et 50$ CA par transaction. Vous pouvez consulter notre grille tarifaire complète dans la section "Tarifs" de notre site web.
